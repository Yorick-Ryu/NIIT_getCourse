1.代码内容介绍  
------

[Cheerio官方文档翻译](https://juejin.cn/post/6844904135767097352#heading-37 "悬停显示")<br><br>

[getClass.js](/NIIT_getCoursegetClass.js)是可运行在本地的核心代码，是[Paser.js](/NIIT_getCourse/Paser.js)的基础<br>
